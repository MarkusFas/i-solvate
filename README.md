## i-solvate

i-solvate is a software tool designed to facilitate the solvation of molecular structures for computational chemistry simulations. It provides an easy-to-use interface for adding solvent molecules around a target molecule, ensuring proper orientation and distribution based on specified parameters.

### Features
- User-friendly command-line interface
- Supports various solvent models (e.g., water, ethanol, etc.)
- Customizable solvation box dimensions
- Automatic placement of solvent molecules
- Compatibility with popular molecular dynamics software formats
